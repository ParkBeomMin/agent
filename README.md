# 노마드코더 Agent

## 초기 셋팅

1. 프로젝트 생성
```
uv init [project name]
```
2. 프로젝트 의존성 설치
```
uv sync
```
```
uv add [패키지명]
```
3. 플러그인 설치
[jupyter|https://marketplace.cursorapi.com/items/?itemName=ms-toolsai.jupyter], [python|https://marketplace.cursorapi.com/items/?itemName=ms-python.python]
4. 실행
```
uv run main.py
```
